hexo d command rewrite the chame file on github pages

hexo d 命令会重写 github pages的chame文件

当你在github pages上设置自己的域名的时候github会在你的仓库里创建chame文件，但hexo d命令会覆盖chame文件导致域名的设置丢失。

所以你应该在./source文件夹中添加CHAME文件其中包含你设置的私有域名，这样在执行hexo g命令和hexe d时chame文件会被部署到仓库中。
