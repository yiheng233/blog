Add-kali-source-to-ubuntu

电脑坏了！！！别学

把kali的源添加到ubuntu中

1.添加源

把kali的源添加到 /etc/apt/sources.d

kali 官方文件 https://www.kali.org/docs/general-use/kali-linux-sources-list-repositories/

2.添加kali-archive-keying.gpg

去kali里面拷一个密钥然后 sudo apt-key add [path of kali-archive-keying.gpg]

可以参考一下 https://qastack.cn/ubuntu/291035/how-to-add-a-gpg-key-to-the-apt-sources-keyring

3. sudo apt update

然后ubuntu就会被滚成kali开不了机了

(包冲突时可以sudo dpkg -P [package name] 原贴 https://askubuntu.com/questions/176121/dpkg-error-trying-to-overwrite-file-which-is-also-in)

run-x86-on-arm64-ubuntu

在arm64 ubuntu中运行 x86-64 程序

1.抄kali arm的官方文档

  • 有b站视频如何在 M1 Arm Mac 上调试 X86 Linux 应用 https://www.bilibili.com/BV12B4y1i7dc

  • kali的官方文档Running x86 code on ARM devices https://www.kali.org/docs/arm/x86-on-arm

这里就粘一下kali的命令

2.调整sources.list

因为ubuntu arm用的源与ubuntu amd64用的源不一样所以要指定每一个源的架构
askububtu上的回复 https://askubuntu.com/questions/1211022/how-to-install-i386-amd64-packages-on-arm-or-any-other-arch-from-ubuntu-ports
在源地址前’deb’后面加[arch= yourArchitecrure]
然后就可以愉快的sudo apt update了
